# LTA - Live Tango App

PWA con WebRTC para transmisiones en vivo anónimas.

## Setup

```bash
npm install
npm run dev
```

Accede a `http://localhost:3000`

## Estructura

- `app/` - Páginas Next.js
- `lib/` - Hooks y utilidades
- `components/` - Componentes React
- `public/` - Assets estáticos

## Dev

```bash
npm run dev    # localhost:3000
npm run build  # Build
npm start      # Producción
```
